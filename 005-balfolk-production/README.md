# Project 005: Balfolk Markov Jig (16 Measures)

This project demonstrates the integration of the three-agent Musicom ecosystem.

## Collaboration Details
- **Logic**: Derived from `musicom_ai`'s Markov Chain approach, applied to a G-Major Balfolk scale.
- **Synthesis**: Produced using `musicom-agent`'s calibrated acoustic engines (Violin and Guitar).
- **Structures**: Follows the `musicom_research` standards for pulse-based 6/8 meters.

## Stats
- **Genre**: Balfolk (Cercle Circassien)
- **Meter**: 6/8
- **Length**: 16 Measures
- **BPM**: 118
