# Musicom Projects

Repository purpose: generated and curated Musicom composition projects.

This repo stores project folders, MIDI, audio previews, analysis, dashboards, notes, and project-local scripts.

## Structure

- `000-project-template/` — template for new projects.
- `NNN-project-name/` — one project per numbered folder.
- `index.html` — portfolio dashboard for GitHub Pages.
