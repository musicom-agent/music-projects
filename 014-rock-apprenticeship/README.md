# 014 — Rock Apprenticeship

Concept: learn rock by composing short studies.

Core method: riff → groove → power-chord harmony → song form → arrangement.

## Current studies

### Study 01

- Title: First Riff Engine
- Genre focus: classic hard rock / garage rock
- Key: E minor / E blues
- Tempo: 128 BPM
- Meter: 4/4
- Form: Intro x2, Verse x4, Chorus x4, Outro x2

### Study 02

- Title: Blues Rock Shuffle Engine
- Genre focus: blues rock / shuffle boogie
- Key: E blues
- Tempo: 112 BPM
- Meter: 4/4 with triplet shuffle subdivision
- Form: 24 bars: 12-bar blues x2
- Files: `MIDI/study-02-blues-rock-shuffle-engine.mid`, `Audio/study-02-blues-rock-shuffle-engine.ogg`

### Study 03

- Title: Voice-Derived Motif
- Source: incoming Telegram voice/sound
- Analysis: stable pitch around D5 (~590 Hz), high brightness, clear onsets
- Composition move: treat D5 as b7 in E blues, then resolve through B–Bb–A–G–E
- Tempo: 108 BPM
- Files: `MIDI/study-03-voice-derived-motif.mid`, `Audio/study-03-voice-derived-motif.ogg`

## Rock rules we test

1. Guitar is rhythm engine, not only harmony.
2. Riff must survive loop repetition.
3. Drums create backbeat: snare on beats 2 and 4.
4. Bass often doubles root/riff, locking kick.
5. Power chords hide major/minor, so melody decides mood.
6. Verse can be tight/low; chorus wider/louder.
